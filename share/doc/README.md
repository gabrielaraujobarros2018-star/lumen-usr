# Lumen

Lumen is a full-featured, user-facing Linux operating system built around freedom, privacy, and long-term stability.

It is not a minimal or lightweight experiment.
It is a complete operating system designed to be used daily.

Current footprint: ~6GB (and growing with refinement, not bloat).

---

# Philosophy

Lumen is built on three foundational principles:

## 1. Freedom

* Open technologies
* Transparent system behavior
* No forced ecosystem lock-in
* User control over configuration and updates

## 2. Privacy

* No telemetry by default
* Local-first design philosophy
* Strong repository integrity
* Secure-by-design architecture

## 3. Stability

* Structured release progression
* Controlled updates
* Verified packages
* Clear architectural layering

Lumen prioritizes predictable behavior over trend-driven experimentation.

---

# What Lumen Is

Lumen is:

* A full desktop-class Linux distribution
* Built on GNOME core technologies
* Powered by Wayland
* Using Phosh as its shell interface
* Designed with convergence in mind (desktop + touch adaptability)

It includes a complete system stack:

* Modern graphical environment
* Full package management
* Core productivity tools
* Networking and firewall configuration
* Security-ready installation paths

This is not a stripped-down environment.
This is a complete system.

---

# Technical Direction

Lumen is architected with intentional layering:

1. Firmware (UEFI)
2. Bootloader
3. Linux Kernel
4. systemd init system
5. Core system services
6. Wayland compositor
7. Phosh shell
8. GNOME core components
9. Applications

Each layer is modular, auditable, and replaceable where appropriate.

---

# Why Not Lightweight?

Lumen does not aim to be minimal for the sake of numbers.

It aims to be:

* Complete
* Capable
* Reliable
* Usable out-of-the-box

A modern operating system should not require heavy manual assembly to feel finished.

Lumen ships full — not fragile.
