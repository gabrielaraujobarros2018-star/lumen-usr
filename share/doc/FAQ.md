# Lumen

Lumen is a full-featured, user-facing Linux operating system built around freedom, privacy, and long-term stability.

It is not a minimal or lightweight experiment.
It is a complete operating system designed to be used daily.

Current footprint: ~6GB (and growing with refinement, not bloat).

---

# Philosophy

Lumen is built on three foundational principles:

## 1. Freedom

* Open technologies
* Transparent system behavior
* No forced ecosystemn lock-in
* User control over configuration and updates

## 2. Privacy

* No telemetry by default
* Local-first design philosophy
* Strong repository integrity
* Secure-by-design architecture

## 3. Stability

* Structured release progression
* Controlled updates
* Verified packages
* Clear architectural layering

Lumen prioritizes predictable behavior over trend-driven experimentation.

---

# What Lumen Is

Lumen is:

* A full desktop-class Linux distribution
* Built on GNOME core technologies
* Powered by Wayland
* Using Phosh as its shell interface
* Designed with convergence in mind (desktop + touch adaptability)

It includes a complete system stack:

* Modern graphical environment
* Full package management
* Core productivity tools
* Networking and firewall configuration
* Security-ready installation paths

This is not a stripped-down environment.
This is a complete system.

---

# Technical Direction

Lumen is architected with intentional layering:

1. Firmware (UEFI)
2. Bootloader
3. Linux Kernel
4. systemd init system
5. Core system services
6. Wayland compositor
7. Phosh shell
8. GNOME core components
9. Applications

Each layer is modular, auditable, and replaceable where appropriate.

# FAQ

## Is Lumen a lightweight distribution?

No.

Lumen is intentionally full-featured. At ~6GB, it ships with a complete desktop environment, core tools, and a structured system stack. It prioritizes completeness and usability over minimal footprint.

---

## Why is Lumen focused on GNOME, Wayland, and Phosh?

Lumen builds on modern, actively maintained technologies.

* GNOME provides a mature ecosystem and consistent design language.
* Wayland offers improved security and a modern display protocol.
* Phosh enables touch-ready, convergence-focused interaction.

This combination supports both desktop stability and adaptive interface goals.

---

## Does Lumen include telemetry?

No.

Lumen follows a privacy-first philosophy. There is no telemetry enabled by default. System behavior is transparent and user-controlled.

---

## Who is Lumen for?

Lumen is for users who want:

* A daily-driver Linux system
* Strong privacy defaults
* A structured and stable release model
* A complete environment out-of-the-box

It is not targeted at extreme minimalists or experimental tinker-only setups.

---

## Is Lumen ready for daily use?

Lumen is currently in early development phases. Developer Preview builds are designed for testing and validation.

Stable releases will be clearly marked and are intended for daily usage once declared.

---

## What makes Lumen different from other distributions?

Lumen differentiates itself through:

* A strict stability discipline before stable releases
* A mobile-first convergence mindset
* A refusal to chase trends at the cost of predictability
* A full-featured system without forced ecosystem control

It grows deliberately rather than rapidly.

---

## When will Stable 1.0 release?

There is no deadline-based release schedule.

Stable 1.0 will be released when the system meets defined stability, performance, and reliability criteria. Proof determines release — not time.

---

Yes.

Lumen advances through consistent refinement and community involvement.
